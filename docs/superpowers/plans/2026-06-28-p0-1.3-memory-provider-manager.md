# ¬ß1.3 MemoryProvider + MemoryManager Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:executing-plans (single-agent inline, per `CLAUDE.md` ¬ß5). Steps use checkbox (`- [ ]`) syntax.

**Goal:** Port Hermes's `MemoryProvider` ABC + `MemoryManager` orchestrator + `<memory-context>` fence, add a minimal built-in provider over the ¬ß1.2 store, and attach it to the ¬ß1.1 loop through the existing `MemoryHooks` seam ‚Äî with **no change to `agent_loop.py`**.

**Architecture:** Faithful port of `agent/memory_provider.py` + `agent/memory_manager.py` (trimmed of Hermes-only couplings), a `BuiltinMemoryProvider` wrapping the ¬ß1.2 `MemoryStore`, and a `MemoryManagerHooks` adapter that implements `core/hooks.py::MemoryHooks` by delegating to a `MemoryManager`. A `composition.py` helper assembles them and fills the ¬ß1.1 `memory_snapshot` / `provider_block` slots. Read/lifecycle path only; the governed model-facing write tool is ¬ß1.5.

**Tech Stack:** Python ‚â•3.11, stdlib `concurrent.futures`/`threading`/`re`, pytest. No new deps.

```python
import time, threading
from jobpin_agent.memory.manager import MemoryManager, _SYNC_DRAIN_TIMEOUT_S

class FakeProvider:
    def __init__(self, name="ext", recall="", tools=None, sleep=0.0, raise_on=None):
        self._name, self._recall, self._tools = name, recall, tools or []
        self._sleep, self._raise_on = sleep, raise_on or set()
        self.synced = []
    @property
    def name(self): return self._name
    def is_available(self): return True
    def initialize(self, session_id, **kw): pass
    def system_prompt_block(self): return f"[{self._name} block]" if self._name != "builtin" else ""
    def prefetch(self, query, *, session_id=""):
        if "prefetch" in self._raise_on: raise RuntimeError("boom")
        return self._recall
    def queue_prefetch(self, query, *, session_id=""): pass
    def sync_turn(self, user, assistant, *, session_id="", messages=None):
        if self._sleep: time.sleep(self._sleep)
        self.synced.append((user, assistant))
    def get_tool_schemas(self): return self._tools
    def handle_tool_call(self, name, args, **kw): return '{"ok": true}'
    def on_pre_compress(self, messages): return f"fact:{self._name}"
    def on_session_switch(self, new, *, parent_session_id="", reset=False, **kw): pass
    def on_delegation(self, task, result, *, child_session_id="", **kw): pass
    def shutdown(self): pass
```
